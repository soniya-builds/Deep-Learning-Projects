# Deep Learning Projects

This repository contains a collection of **Deep Learning projects** developed as part of my learning journey in Artificial Intelligence and Machine Learning.

The projects focus on applying deep learning techniques to solve real-world problems such as **image classification, computer vision, and model deployment**.

Currently, the repository includes:

- **Potato Disease Classification using CNN**
  - Image classification using Convolutional Neural Networks
  - Detects potato leaf diseases from images
  - Model served using FastAPI

More deep learning projects will be added as I continue exploring new models, architectures, and applications.
